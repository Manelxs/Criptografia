# 🔐 Simulador del Cifrado de Alberti en Python

Este repositorio contiene una implementación moderna y estructurada en Python del **Cifrado de Alberti**, uno de los primeros sistemas de cifrado polialfabético de la historia, inventado por el polímata italiano Leon Battista Alberti en 1467.

La aplicación simula el comportamiento del famoso disco criptográfico físico utilizando dos alfabetos (fijo y móvil desordenado), combinándolo con una interfaz interactiva de consola que genera llaves de acceso únicas y gestiona un historial de mensajes.

---

## ✨ Características principales

* **Fidelidad Histórica:** Simulación exacta de los discos concéntricos de Alberti, empleando un alfabeto interno desordenado y cambios periódicos de posición (polialfabetismo).
* **Gestión de Sesión Interactiva:** Menú en consola fácil de usar para encriptar, desencriptar y listar elementos.
* **Tokens Únicos de Recuperación:** Cada mensaje se encripta y se asocia automáticamente a una llave aleatoria única de 6 dígitos.
* **Historial Cronológico:** Una opción dedicada para listar todas las palabras cifradas de arriba hacia abajo con numeración ascendente automática.
* **Buenas Prácticas:** Código modularizado en Programación Orientada a Objetos (POO), manejo de caracteres especiales/espacios y nombrado limpio de variables.
